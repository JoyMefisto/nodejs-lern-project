# nodejs-lern-project
Project on Express.js


#### Install
- `cd nodejs-lern-project`
- `npm install`

#### Run
- `cd nodejs-lern-project`
- `npm run dev`
- `localhost:3000`


##### Routers
```
- home: /
- search: /search
- about: /about
- users: /users
- user: /user/1
- 404: /404
- 500: Server Error
```
